## 三、課程目標與特色（表格3）

### 班級大綱

```
AI 輔助程式開發的時代已經來臨！Google 最新推出的 Antigravity 工具，讓沒有程式背景的人也能透過自然語言描述，讓 AI 幫你撰寫完整的應用程式。這就是「Vibe Coding」—— 用感覺來寫程式！

本課程專為非程式背景人士設計，帶領學員從零開始，透過對話式指令操作 Antigravity，快速打造 4 大超實用自動化流程。您將學會如何建立專屬 Skills 與 Workflow、運用 MCP（Model Context Protocol）擴充工具，甚至開發 YouTube 字幕產生器等進階應用。

無論您是產品經理、設計師、行銷人員、業務主管，或是任何想進入程式開發領域的初學者，本課程都將帶您體驗全新的開發方式，真正實現「人人都能開發」的願景。
```

### 班級目標

```
1. 理解 Vibe Coding 概念與 AI 輔助程式開發的工作流程
2. 掌握 Antigravity 工具的操作方式與各項參數設定
3. 能夠建立專屬的 Skills 與 Workflow 自動化流程
4. 學會透過 MCP 協議擴充 AI 能力，整合外部工具
5. 完成 4 大實用專案：資料整理腳本、文件轉換Skill、頻道分析報告、字幕產生器
```

### 培訓評價目標

```
勾選：L2
```

### 班級對象

```
1. 對程式開發有興趣但沒有程式基礎的職場人士
2. 產品經理、專案經理、企劃人員想了解開發流程
3. 設計師、行銷人員希望能自行製作簡易工具
4. 中小企業主或創業者想快速製作 MVP 產品
5. 對 AI 輔助開發感興趣的創新實踐者
```
